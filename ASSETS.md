# Project Assets

This document provides the link to all necessary files for this project, including the Streamlit application code and trained machine learning model.
https://drive.google.com/drive/folders/1Td038fsQ6JzOuX5PmBd_gEW9mSSmzlBZ?usp=sharing

## Application Files

- **`streamlit_app.py`**: The main Streamlit application code.
  
- **`requirements.txt`**: List of Python dependencies for the Streamlit app.

## Model and Data Assets

- **`best_rf_model_tuned.pkl`**: The trained Random Forest Regressor model.
  
- **`training_columns.pkl`**: Column names used during model training to ensure correct input features.
  
- **`feature_importances.pkl`**: Dataframe containing feature importances for visualization.
  
- **`y_train_mean.pkl`**: Mean of the training target variable for contextual plots.
  
- **`y_train_std.pkl`**: Standard deviation of the training target variable for contextual plots.
  
---

**Note:** If you are running the Streamlit app locally, please download all files and place them in the same directory as `streamlit_app.py`.
